# 🍅 Tomato Disease Diagnosis System

## Explainable Tomato Disease Detection, Severity Assessment and Crop Advisory System

An AI-powered system that detects tomato leaf diseases, explains predictions using Grad-CAM, estimates disease severity, provides nutrient recommendations, and generates PDF health reports.

---

## Features

* Disease Detection using EfficientNetB0
* Grad-CAM Explainability
* Disease Severity Analysis
* Crop Health Assessment
* Nutrient Deficiency Advisor
* PDF Report Generation
* Dashboard Analytics
* Streamlit Web Application

---

## Model Performance

* Model: EfficientNetB0
* Accuracy: 96.88%

---

## Diseases Detected

* Tomato Early Blight
* Tomato Late Blight
* Tomato Leaf Mold
* Tomato Healthy
